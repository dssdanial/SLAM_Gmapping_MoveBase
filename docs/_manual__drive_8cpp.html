Detailed Description

Allows manual control of the robot using keyboard input.

The node enables the user to move the robot in the environment using the keyboard. The velocity and direction of the robot are managed by four variables, two for velocity values (linear and angular) and two for direction (linear and angular).

void get_input ( )

Captures keyboard input and computes velocity commands.

This function prints the user instruction menu for keyboard commands, captures user keyboard input, computes the new velocity and direction commands, and then publishes a Twist message with these commands to the /cmd_vel topic.
